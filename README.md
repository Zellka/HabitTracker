# Habit
![](https://github.com/Zellka/HabitTracker/blob/master/image/ic_app.png)

## Функции
* Просмотр привычек по типу(хорошие, плохие)
* Добавление/изменение/удаление привычки
* Фильтрация(по названию, дате, приоритету)
* Поиск по списку
* Боковая навигация
